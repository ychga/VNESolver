配置好环境后，根目录打开命令行执行下面的命令即可将项目作为API后端运行：
uvicorn app:app --reload
端口号与url地址见app.py中的定义

主要版本：
* Python 3.9
* PyTorch 1.12.1
* CUDA 11.3
* NVIDIA GPU
详细环境参见environment-lock.yaml

使用conda创建环境：
conda env create -f environment.yml
conda activate virne

使用conda创建版本号严格一致的环境：
conda env create -f environment-lock.yml
